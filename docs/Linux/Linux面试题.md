# 查找文件有哪些常用命令

- 
- **grep -r 字符串 路径**



1. 查找包含指定单词的文件

   ```sh
   grep -rwi 目录 -e "hello"
   # r:递归子目录，查询当前目录下所有子目录
   # w：word，单词，单词的范围是特殊符号分割出的字符串
   # i：不区分大小写
   ```

   

2. 查找包含指定非单词的文件

   ```sh
   grep -Ri /home/sharplee -e "你好"
   # 去掉w 单词即可
   ```

# 查询包含指定字符串的文件

有多个linux文件，需要查找那个文件中包含指定的字符串的文件名

```sh
grep -ril "指定的字符串" *
```

查找根目录下的文件包含指定的字符串的对应内容

```sh
find / -name "*" | xargs grep -i "指定的字符串"
```
