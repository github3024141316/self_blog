# 查看文件有哪些命令

```lua
vi 文件名   #编辑方式查看，可修改

cat 文件名  #显示全部文件内容

more 文件名 #分页显示文件内容

less 文件名 #与 more 相似，更好的是可以往前翻页

tail 文件名 #仅查看尾部，还可以指定行数

head 文件名 #仅查看头部,还可以指定行数
```



# 查询包含指定字符串的文件

有多个linux文件，需要查找那个文件中包含指定的字符串的文件名

```sh
grep -ril "指定的字符串" *
```

查找根目录下的文件包含指定的字符串的对应内容

```sh
find / -name "*" | xargs grep -i "指定的字符串"
```

# 如何查看进程的内存占用

```sh
cat /proc/[pid]/status
```
